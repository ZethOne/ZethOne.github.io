Inicio

Este curso está diseñado para introducirte al mundo de la automatización de pruebas de software, tanto en aplicaciones web como en APIs. Aprenderás los fundamentos herramientas como Playwright, GitHub y Github Actions, asi como buenas prácticas para convertirte en un profesional del testing automatizado.

Temario

  • Comenzando con Playwright
  • Requerimientos de software
  • Instalación y configuración:
  • Terminologías y conceptos básicos
  • Elementos básicos de la automatización
  • Playwright UI y herramientas
  • Automatización avanzada en Playwright
  • Características adicionales
  • Funciones y manejo de datos
  • Automatización de backend y API mocking
  • Integración con GitHub Actions
